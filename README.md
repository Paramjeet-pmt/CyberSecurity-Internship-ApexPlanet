### Task 1: Foundation & Environment Setup
Cybersecurity fundamentals (CIA Triad, threat types, attack vectors), lab environment setup (VMware, Kali Linux, Metasploitable2), Linux fundamentals, networking basics, cryptography basics, and tool familiarization (Wireshark, Nmap, Burp Suite, Netcat).

### Task 2: Network Security & Scanning
Reconnaissance, port/service scanning with Nmap, vulnerability scanning (OpenVAS/Nessus), packet analysis with Wireshark, and firewall basics.

### Task 3: Web Application Security
OWASP Top 10 vulnerabilities on DVWA — SQL Injection, XSS, CSRF, File Inclusion attacks, Burp Suite Advanced usage, and web security headers.

### Task 4: Exploitation & System Security
Penetration testing methodology, exploitation with Metasploit, password attacks (Hydra, John the Ripper), social engineering simulation, malware basics, and system hardening.

### Task 5: Capstone Project & Incident Response
A self-chosen capstone (pentest report, vulnerability assessment, mini SIEM, or phishing simulation) with full project planning, implementation, incident response simulation, and final documentation.
